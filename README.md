# SuColor - Portal de Seguimiento

Aplicacion web para seguimiento de ordenes de taller y panel administrativo.

## Estructura principal

- `src/paginas`: paginas publicas y de administracion.
- `src/componentes`: componentes reutilizables y modulo administrativo.
- `src/ganchos`: hooks de negocio.
- `src/servicios`: clientes HTTP y llamadas a endpoints.
- `src/biblioteca`: utilidades compartidas y cliente de Supabase.
- `src/tipos`: tipos de TypeScript.
- `src/pruebas`: pruebas unitarias e integracion.
- `config`: configuraciones tecnicas (vite, vitest, tailwind, lint/prettier).
- `docs`: documentacion funcional y tecnica.
- `scripts/pruebas-manuales`: scripts de validacion manual.
- `supabase/sql/manuales`: SQL operativo no versionado como migracion.

## Scripts

- `npm run dev`: entorno local.
- `npm run build`: compilacion de produccion.
- `npm run test:run`: pruebas.
- `npm run lint`: validacion de lint.
