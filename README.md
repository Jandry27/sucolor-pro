# SuColor — Portal Cliente

## Variables de Entorno

Crea un archivo `.env` en la raíz del proyecto copiando este archivo y completando con tus datos:

```
VITE_SUPABASE_URL=https://YOUR_PROJECT_ID.supabase.co
```

## Cómo correr el proyecto

```bash
npm install
npm run dev
```

## Ejemplo de URL de seguimiento

```
http://localhost:5173/track/SC-0001?token=TU_TOKEN
```
